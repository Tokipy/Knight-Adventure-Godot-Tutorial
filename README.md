# Mini Platformer - Godot

This is a practice project developed in Godot, based on a tutorial, with the goal of learning the fundamentals of the engine and 2D game development.

## Description

The project consists of a small platformer game where the player can move horizontally, jump between platforms, interact with enemies.
As a personal modification, a double jump mechanic was implemented, adding extra functionality to the player character.

## Purpose

This project was created as a learning exercise to:
- Understand the structure of a Godot project
- Learn how to use nodes and scenes
- Implement basic movement and physics mechanics

## Assets

The graphical and audio assets used in this project are not original and were obtained from free resources for educational use.
Source: https://brackeysgames.itch.io/brackeys-platformer-bundle

## Project Status

Completed as a practice project. Possible future improvements include visual effects, gameplay enhancements, and additional mechanics.

## 🇪🇸
